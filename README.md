# Deploying Python app
- Update userdata in launch configuration with the following
- App runs on port 5000

```
yum install -y python3
yum install python-pip
git clone https://github.com/nagarkarv/reskill-service.git
cd reskill-service
chmod +x start.sh
./start.sh
```
